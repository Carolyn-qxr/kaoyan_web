# kaoyan_web

功能描述：
一个考研网站
主要实现院校库、资料库、公告管理、日程管理、后台管理员等功能
admin为管理员
home是网站首页等模块
login是登录模块
school为院校库
test是资料库
task是任务管理模块

一个完整的vue项目，运行可能需要安装相关依赖等

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
